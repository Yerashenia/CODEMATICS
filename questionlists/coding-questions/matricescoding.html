Task: Transpose a matrix using Java

Arrays in java can be used to represent matrices. Given the following 2D array as input into your program, produce the 2D array that is labelled output using your program by swapping the rows with the columns using Java.

Input: [[1 , 2, 3], [5, 6, 7], [9, 10, 11], [13, 14, 15]]

Output: [[1, 5, 9, 13], [2, 6, 10, 14], [3, 7, 11, 15 ]]

Visual Example:

Information:

A matrix is a rectangular array of numbers organised into rows and columns. Matrices provide a method of organising, storing and working with mathematical information. They are often used in areas such as computer graphics. A matrix order is identified by the number of rows and columns it has, in that order. A 2 by 2 (2 x 2) matrix, has two rows and two columns. The transpose of the matrix with dimensions R x C is a matrix C x R for Which result matrix B[c][r] = A[r][c].

If we transpose a matrix it becomes the matrix. The rows of the original matrix become columns of the transposed matrix or we can say that the elements of the matrix have been flipped over its main diagonal: The main diagonal is the line going from the top left element to the bottom right element of the square matrix.
