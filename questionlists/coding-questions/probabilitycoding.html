Task: Construct a program in Java that calculated the probability of getting at least 2 tails when a coin is tossed 4 times.

Information:

The probability of an event A occurring is written as P(A). P(A) = number of times the event A can occur / total number of possible outcomes. The overall probability of event A occurring and then also event B occurring is P(A) * P(B).

Imagine a coin is tossed three times, how would we find the probability of getting at least 2 tails. To solve this problem in maths we need to find the sample space S, and in order to do this we are going to create a tree diagram.

Therefore S = (HHH, HHT, HTH, HTT, THH, THT ,TTH, TTT)

We can see from S that there are 4 possible outcomes where we would get at least 2 tails. However, if we wanted to find of getting at least 2 tails when a coin is tossed 4 times, the process becomes very tedious.
